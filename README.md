# Tantrix
Implementation of the Tantrix puzzle game in Python.

## Launch tantrix on one computer
#### Clone the master branch:
git clone https://github.com/aless80/tantrix.git
#### Start tantrix.py
cd tantrix
python tantrix.py

## Launch server and two clients
cd ~/tantrix/

python ./server.py

python ./tantrix.py
